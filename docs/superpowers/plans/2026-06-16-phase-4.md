# Phase 4 — Verification + Data Lifecycle

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development (recommended) or superpowers:executing-plans. Steps use `- [ ]` checkbox tracking.

**Goal:** close the bootstrap lifecycle. Real subprocess verify engine with full terminal state machine, `cognit redaction test`, `cognit gc` (artifact archive + db size), `cognit export/import` (lossless tar.gz round-trip). A session can run, verify, redact-test, clean up, and move.

**Architecture:** 4 sub-beads fan out in parallel after 1 scaffold slice. 4a is biggest (subprocess engine). 4b/4c/4d are independent CLI slices. 4e is E2E + docs. New dep: `tar` (node-tar). No new DB tables; extends existing schemas via a `1.1.0` migration (artifact indexing, payload field additions).

**Tech Stack:** unchanged — Node 24, TS 5.5+, Effect, Vitest, better-sqlite3. Add `tar@7`. New package: `packages/verification` flips from stub to real.

- [ ] **Step 2: Implement `spawn.ts`**

```ts
import { spawn } from "node:child_process"
import { Effect } from "effect"

export type SpawnError = { _tag: "SpawnError"; code: "enoent" | "eacces" | "eperm" | "other"; message: string }

export const spawnVerification = (opts: {
  command: readonly string[]
  cwd: string
  env: NodeJS.ProcessEnv
  signal: AbortSignal
}): Effect.Effect<{ exitCode: number; stdout: string; stderr: string; durationMs: number; error?: SpawnError }, SpawnError> =>
  Effect.gen(function* () {
    const started = Date.now()
    const [cmd, ...args] = opts.command
    const child = spawn(cmd, args, { cwd: opts.cwd, env: opts.env, signal: opts.signal })
    let stdout = "", stderr = ""
    child.stdout.on("data", d => { stdout += d.toString() })
    child.stderr.on("data", d => { stderr += d.toString() })
    const code = yield* Effect.tryPromise({
      try: () => new Promise<number | null>((resolve, reject) => {
        child.on("error", err => {
          const e = err as NodeJS.ErrnoException
          if (e.code === "ENOENT") reject({ _tag: "SpawnError", code: "enoent", message: e.message })
          else if (e.code === "EACCES") reject({ _tag: "SpawnError", code: "eacces", message: e.message })
          else if (e.code === "EPERM") reject({ _tag: "SpawnError", code: "eperm", message: e.message })
          else reject({ _tag: "SpawnError", code: "other", message: e.message })
        })
        child.on("close", c => resolve(c))
      }),
      catch: e => e as SpawnError,
    })
    return { exitCode: code ?? -1, stdout, stderr, durationMs: Date.now() - started }
  })
```

```ts
import { join } from "node:path"

export const sha256 = (text: string): string =>

export const writeArtifact = (opts: {

  text: string
```

```ts
import { truncateExcerpt } from "./capture.js"
import { writeArtifact, sha256 } from "./artifact.js"

export type RunVerificationInput = {
  readonly command: readonly string[]
  readonly type: "lint" | "typecheck" | "test" | "build" | "benchmark" | "custom"
  readonly cwd: string
  readonly env: NodeJS.ProcessEnv
  readonly signal: AbortSignal
  readonly projectPaths: { artifacts: string }
  readonly onTerminal: (event: {
    type: "verification_passed" | "verification_failed" | "verification_errored"
    payload: Record<string, unknown>
  }) => Effect.Effect<void, never>
}

export const runVerification = (input: RunVerificationInput) =>
  Effect.gen(function* () {
    const result = yield* Effect.either(spawnVerification(input))
    if (result._tag === "Left") {
      const err = result.left
      return yield* input.onTerminal({
        type: "verification_errored",
        payload: { error: err.message, ...(err.code !== "other" ? { error_code: err.code } : {}) },
      })
    }
    const { exitCode, stdout, stderr, durationMs } = result.right
    const stdoutExcerpt = truncateExcerpt(stdout)
    const stderrExcerpt = truncateExcerpt(stderr)
    // only write artifact for non-trivial output (>1KB)
    let createdArtifactId: string | undefined
    if (stdout.length + stderr.length > 1024) {
      const ref = yield* writeArtifact({ projectPaths: input.projectPaths, text: stdout + stderr, kind: "terminal-log" })
      createdArtifactId = ref.id
    }
    if (exitCode === 0) {
      return yield* input.onTerminal({
        type: "verification_passed",
        payload: { exit_code: exitCode, duration_ms: durationMs, stdout_excerpt: stdoutExcerpt, stderr_excerpt: stderrExcerpt, ...(createdArtifactId ? { created_artifact_id: createdArtifactId } : {}) },
      })
    }
    return yield* input.onTerminal({
      type: "verification_failed",
      payload: { exit_code: exitCode, duration_ms: durationMs, stdout_excerpt: stdoutExcerpt, stderr_excerpt: stderrExcerpt, ...(createdArtifactId ? { created_artifact_id: createdArtifactId } : {}) },
    })
  })
```

```ts
export const redactWithSpans = (
  text: string,
  patterns: ReadonlyArray<BuiltInRedactionPattern>,
): { redacted: string; hits: Array<{ pattern: string; span: [number, number]; match: string }> } => {
  const hits: Array<{ pattern: string; span: [number, number]; match: string }> = []
  let redacted = text
  for (const p of patterns) {
    const re = new RegExp(p.regex.source, p.regex.flags.includes("g") ? p.regex.flags : p.regex.flags + "g")
    redacted = redacted.replace(re, (match, ...rest) => {
      const offset = rest[rest.length - 2] as number
      hits.push({ pattern: p.name, span: [offset, offset + match.length], match })
      return p.replacement
    })
  }
  return { redacted, hits }
}
```

```ts
export const registerGc = (program: Command) =>
  program.command("gc")
    .option("--dry-run", "print what would happen, do not mutate")
    .option("--force", "skip the confirmation prompt")
    .option("--max-age-days <n>", "override cleanup.artifact_max_age_days", parseInt)
    .action(async (opts) => {

      const config = ctx.config

          await move(a.path, join(ctx.paths.archive, `${a.id}.${a.kind}`))
```

```ts
import { Command } from "commander"
import { createReadStream, createWriteStream } from "node:fs"
import { copyFile } from "node:fs/promises"
import { tmpdir } from "node:os"
import { join } from "node:path"
import { createGzip } from "node:zlib"
import { pipeline } from "node:stream/promises"
import { pack } from "tar"
import { vacuumInto } from "@cognit/db"

const BUNDLE_FORMAT_VERSION = 1

export const registerExport = (program: Command) =>
  program.command("export")
    .requiredOption("--output <path>", "output .tar.gz path")
    .option("--include-artifacts", "include .cognit/artifacts/ in the bundle")
    .action(async (opts) => {
      const ctx = await loadContext(program)
      const work = join(tmpdir(), `cognit-export-${Date.now()}`)
      await mkdir(work, { recursive: true })
      // 1. dump db
      await Effect.runPromise(vacuumInto(ctx.db, join(work, "cognit.db")))
      // 2. copy cognit.yaml
      await copyFile(ctx.paths.config, join(work, "cognit.yaml"))
      // 3. optional artifacts
      const files: string[] = ["manifest.json", "cognit.yaml", "cognit.db"]
      if (opts.includeArtifacts) {
        await copyDir(ctx.paths.artifacts, join(work, "artifacts"))
        files.push("artifacts")
      }
      // 4. write manifest
      await writeFile(join(work, "manifest.json"), JSON.stringify({
        format_version: BUNDLE_FORMAT_VERSION,
        created_at: new Date().toISOString(),
        project_name: ctx.config.project.name,
        schema_version: "1.1.0",
      }))
      // 5. tar.gz
      await pipeline(
        pack({ cwd: work, gzip: false, portable: true }),
        createGzip(),
        createWriteStream(opts.output),
      )
      console.log(`Exported to ${opts.output}`)
    })
```

```ts
import { Command } from "commander"
import { extract } from "tar"
import { Effect } from "effect"
import { readFile, mkdir } from "node:fs/promises"
import { mkdtemp, rm } from "node:fs/promises"
import { tmpdir } from "node:os"
import { join } from "node:path"

export const registerImport = (program: Command) =>
  program.command("import")
    .requiredOption("--input <path>", "input .tar.gz path")
    .option("--merge-strategy <strategy>", "skip|overwrite|fork", "skip")
    .action(async (opts) => {
      const ctx = await loadContext(program)
      const work = await mkdtemp(join(tmpdir(), "cognit-import-"))
      try {
        await extract({ file: opts.input, cwd: work })
        const manifest = JSON.parse(await readFile(join(work, "manifest.json"), "utf8"))
        if (manifest.format_version !== 1) throw new Error(`Unsupported bundle format_version ${manifest.format_version}`)

        // Open the imported DB in a side connection, read all rows
        const importCtx = await openImportedDb(join(work, "cognit.db"))
        const importRows = await readAllRows(importCtx)

        // Per-strategy merge into local DB
        for (const row of importRows) {
          const localExists = await rowExists(ctx, row.table, row.id)
          if (opts.mergeStrategy === "skip" && localExists) continue
          if (opts.mergeStrategy === "overwrite" && localExists) {
            await deleteRow(ctx, row.table, row.id)
          }
          if (opts.mergeStrategy === "fork") {
            row.id = ulid()
            // rewrite FKs
            row.payload.session_id = remapId(row.payload.session_id, importCtx.idMap)
            // etc.
          }
          await insertRow(ctx, row)
        }

        // Copy cognit.yaml fields that aren't already in local config
        const importedConfig = parseCognitYaml(await readFile(join(work, "cognit.yaml"), "utf8"))
        await mergeConfig(ctx.paths.config, importedConfig, opts.mergeStrategy)

        console.log(`Imported ${importRows.length} rows with strategy=${opts.mergeStrategy}`)
      } finally {
        await rm(work, { recursive: true, force: true })
      }
    })
```

`readAllRows` returns rows in dependency order: `projects, sessions, actors, events, snapshots, artifacts, edges, constraint_rules, hypotheses, inbox_processed, schema_version`. The `events` table rows have their `version` field inspected; if != local `schema_version`, call `migratePayload` via the registry (this is why 6bz.10 step 1 is the foundation).

## Risks (from research + new)

- **4a subprocess engine isolation** — `child_process.spawn` is OS-dependent. Mitigation: tests run on Linux/macOS only; Windows `cmd.exe` quirks are a v0.2 follow-up (note in phase-4-results "Windows known-issue").
- **4a exit_code=null on signal kill** — `close` event can fire with `code: null` when the process is killed by signal. Map `null` to a synthetic `exit_code: 128 + signal_number` and emit `verification_cancelled` instead of `verification_failed`. Add a test.
- **4c archive_action moves on-disk file** — if the file is open by a reader, move fails with EBUSY on Linux. Mitigation: advisory close in the CLI; no hard guarantee. Document.
- **4d merge-strategy fork FK remap** — the FK remap must cover all 6 columns + edge from/to. A missed column yields orphan rows. Mitigation: the import code constructs an `idMap: Map<string, string>` from the source DB and applies it as a single pass. Add an E2E that exports→imports→exports and asserts all FKs are intact (no orphan `events.session_id`).
- **4d VACUUM INTO requires the DB not to be in WAL** — actually VACUUM INTO works in WAL mode (SQLite 3.27+). Just verify the better-sqlite3 version. If not, fall back to `.backup()`.
- **4b `extra_patterns` vs `patterns` field-name mismatch** — the spec wording says `extra_patterns` but the actual schema field is `redaction.patterns`. Decision: keep `patterns` (matches the code) and add a one-line note in the CLI help text saying "patterns: extra patterns on top of the built-ins".
- **4e "current_state" vs "state" wording mismatch in plan.xml** — only `HypothesisState` uses `current_state`; `ConclusionState` and `DecisionState` use `state`. The GC code uses `state` (correct). Spec is wrong; do not change code to match.
- **Phase 3 doc "Out of phase 3" lists `cognit redaction test`** — when phase 4 ships, remove that line from a follow-up edit to `docs/phase-3-results.md` (out of phase 4 scope per acceptance lock; defer to phase 4.5 docs cleanup).
